Welcome to WA-Models documentation!
===================================

**WA-Models** is a collection of Python libraries for data enthusiasts, 
with an initial focus on football (soccer) analytics.

The first available module is :doc:`soccerdonna_scraper`, 
a scraper for extracting detailed player and club data from 
`Soccerdonna.de <https://www.soccerdonna.de>`_.

Check out the :doc:`soccerdonna_scraper` section for further information, 
including how to :ref:`installation` and usage examples.

.. note::

   This project is under active development. New features and modules may be added in the future.
